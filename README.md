# 燕云十六声资料展示与江湖探索平台

基于 Vue 3 + Vite + Vue Router 的中文前端展示项目，围绕《燕云十六声》的山河区域、武学奇术、十六声文化和个人武林录进行资料化展示。

## 页面说明

- 首页：沉浸式入口、官方图源轮播、资料模块入口。
- 山河图志：展示清河、开封、河西、不见山，以及开封皇宫扩展空间。
- 武学图鉴：围绕兵器、奇术、流派、版本新增内容，支持搜索、筛选、详情查看与收藏。
- 十六声文化：展示文化索引、主题详情和“文明之声”阅读线索。
- 我的武林录：读取本地收藏，展示已入卷武学和个人资料索引。

## Vue 技术点

- 组件化页面拆分
- Vue Router 路由导航
- `v-for` 列表渲染
- `v-if` 条件渲染
- `computed` 派生数据
- `ref` 响应式状态
- `watch` 搜索提示与收藏同步
- `props` 父子组件传值
- `emits` 子组件事件通知
- `Transition` 切换动画
- `localStorage` 本地收藏持久化

## 素材说明

项目图片来源于《燕云十六声》官方公开页面和官方新闻页。本项目仅作学习展示用途，不用于商业用途；游戏名称、相关素材与设定归官方权利方所有。
